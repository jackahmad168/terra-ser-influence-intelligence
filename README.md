# Terra Ser Influence Intelligence

This repository contains the initial project structure for the Terra Ser Influence Intelligence platform.

## Structure

- backend/ - Node.js API
- frontend/ - React.js dashboard
- ai/ - Python ML scripts
- database/ - SQL schema and seed data
- docs/ - Documentation

## Render deployment

Set the following environment variable in Render:

- DATABASE_URL: your PostgreSQL connection string

The backend is configured to read DATABASE_URL automatically.
